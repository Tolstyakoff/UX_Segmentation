# UX_Segmentation
Варианты лабораторной работы
Для оценки предпочтений пользователей сайта и проведения сегментации разработана анкета. 
Каждый вопрос пользователь оценивает по 5 балльной шкале, от 1 до 5, где  1 - совершенно не удовлетворен, 5 - полностью удовлетворен

Оцените насколько Вас удовлетворяет сайт по следующим критериям
1. Продуманный дизайн без лишних элементов
2. Цветовая палитра сайта
3. Читабельность и размер шрифтов
4. Понятная навигация
5. Наличие "главных" кнопок (купить/заказать)
6. Настроена навигационная цепочка (хлебные крошки)
7. Логичность расположения контента
8. Содержание и контент сайта
9. Уникальные сервисы 
10. Наличие подсказок и индикаторов
11. Скорость загрузки
12. Соответствие мобильной и десктопной версий
13. Возможность настройки сайта под себя

Далее идет блок вопросов по полу и возрасту респонедента

Задача провести сегментацию клиентсткой базы методом кластерного анализа, определить состав и характеристики сегмента, 
выяснить наличие закономерностей между сегментами и половозрастными характеристиками
сформировать визуализированный отчет про проделанной работе
